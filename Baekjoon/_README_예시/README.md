# [Silver IV] 1057_토너먼트

[문제 링크](https://www.acmicpc.net/problem/1057) 

### 🛠️ 분류
수학, 브루트포스 알고리즘

### 💡 접근 방법
a와 b(김지민과 임한수)가 만나기 전까지 항상 이긴다고 가정한다고 했으므로, a와 b의 값을 다음 라운드로 진출했을 때의 번호로 계속 업데이트 하면서, a와 b가 만났을 때, 올라온 라운드의 횟수를 출력하면 된다.

아래의 Key Code는 다음 라운드로 갔을 때의 번호를 출력하는 코드이다.
- 1,2 -> 1
- 3,4 -> 2
- 5,6 -> 3
- ...

두 사람이 만나지 않는 경우는 존재하지 않으므로, 예외 처리는 따로 하지 않았다. 예외처리를 한다고 가정하면, log2(n)보다 큰 자연수만큼 반복문을 돌린 후, 반복문이 끝났을 경우, 두 값이 만나지 않을 경우, -1로 출력하면 될 것이다.
### 🗝️ Key Code (핵심 코드)
```python
def tournament(a: int):
    return (a + 1) // 2
